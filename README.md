SHOE INVENTORY MANAGEMENT SYSTEM

PROJECT OVERVIEW
This project is a python-based inventory management system. It allows users to manage shoe stock by reading data from a file, adding new shoes, searching for shoes, restocking low inventory, and calculating the value of stock. It is designed for retail stores or warehouse stock management.

WHAT THIS PROGRAM DOES
This program allows the user to:
⦁	Load shoe data from a file(inventory.txt)
⦁	Search for a shoe using it's code
⦁	Restock the shoe with the lowest quantity
⦁	Calculate the total value of each shoe(cost x quantity)
⦁	Display the shoe with the highest quantity as "on sale"
⦁	Add new shoes to the inventory
⦁	Automatically update the inventory file
